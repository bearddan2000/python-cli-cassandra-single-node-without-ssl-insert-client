# python-cli-cassandra-single-node-without-ssl-insert-client

## Description
POC for single node without ssl cassandra in python.

- Features
  - single node cassandra
  - cql
    - create keyspace
    - create table
    - single insert
  - pandas dataframe

## Tech stack
- python
  - cassandra-driver
  - padas
- cassandra

## Docker stack
- python:3.8-bullyseye
- cassandra:4.0

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
